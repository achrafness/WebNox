About WebNoX

An educational platform dedicated to web application security training, designed by ESTIN engineering students.

Our Mission

WebNoX aims to build a modern, interactive, and secure e-learning system where students can learn cybersecurity concepts and practice them through realistic labs, integrated challenges, and a gamified progression system.

We believe that hands-on experience is the best way to learn security. Our platform provides a safe environment to practice real-world vulnerabilities without the risks associated with testing on live systems.

Platform Features

Structured Courses

Comprehensive courses covering OWASP Top 10, XSS, SQL Injection, CSRF, SSRF, IDOR, and more.

Hands-on Labs

CTF-style challenges with realistic vulnerable applications. Capture flags and earn points.

Gamification

Earn points, climb the leaderboard, and track your progress through badges and achievements.

Vulnerability Types Covered
  • Cross-Site Scripting (XSS)
  • SQL Injection
  • Cross-Site Request Forgery (CSRF)
  • Server-Side Request Forgery (SSRF)
  • Insecure Direct Object References (IDOR)
  • Authentication Bypass
  • Command Injection
  • File Upload Vulnerabilities
  • XML External Entity (XXE)
  • And many more...

Our Team
